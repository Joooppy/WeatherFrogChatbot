Welcome to the WeatherFrog Chatbot by Joop Timm.


The chatbot predicts the weather for up to 4 days into the future and can predict the weather for all major city.

It is built with the Rasa CALM Framework and additionally powered by GPT-3.5-Turbo as a LLM Command Generator.

To get current weather data the Chatbot is powered by OpenWeatherMap API.


The installation guide is seperate in the InstallationGuide pdf.

A general overview of the project with explanations and reflections is found in WeatherFrogAbstract.pdf


Thank you very much and have fun using my Chatbot!
